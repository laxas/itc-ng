Das Institut zur Qualitätsentwicklung im Bildungswesen betreibt auf diesen Seiten eine Pilotanwendung für den Standard OpenCBA. Es handelt sich dabei um eine Initiative von Institutionen zur Entwicklung eines modularen Systems zum computerbasierten Testen.

Um sich für einen bestimmten Test anzumelden, geben Sie bitte den Namen und das Kennwort für einen Tests ein. Sie können über das Seitenmenü oben rechts weitere Funktionen aufrufen.

Derzeit sind Sie am System angemeldet. Über das Menü oben rechts können Sie zur Startseite eines Tests gelangen.

Bei Fragen zu diesem System wenden Sie sich bitte an Martin Mechtel.
